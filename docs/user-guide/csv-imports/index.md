# CSV Imports

Use CSV imports when you want to prepare files manually and upload products or relationships in batches.

## Available guides

- [Product catalog import](./products.md)
- [Product image upload](./images-upload.md)
- [Shop The Look import](./shop-the-look.md)

## Before you start

- always use the latest template downloaded from your dashboard
- check **Spreadsheet Fields** if you need help with field format
- if special characters look wrong after export, open the file in Google Sheets and download the CSV again before uploading
