# Import Products

There are two ways to import products into FashionAI.

## Option 1: CSV import

Use this option when you want to prepare a file manually and upload products in batches.

- [Open the CSV imports area](../csv-imports/index.md)

## Option 2: direct integration

For some stores, we can pull products directly through a platform integration, without preparing a CSV file manually.

- [See an example of integration-based import](../integrations/vtex.md)

## Which option should you use?

- Choose **CSV import** if your team wants full control over the file before uploading
- Choose **direct integration** if your catalog is already managed in your platform and you want a direct sync flow
