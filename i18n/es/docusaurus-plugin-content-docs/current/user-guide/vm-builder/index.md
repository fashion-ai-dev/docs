# VM Builder

VM Builder es la herramienta que te permite crear y gestionar vitrinas personalizadas de productos para tu sitio web.

## ¿Qué puedes hacer?

Con VM Builder puedes:

- **Crear Colecciones**: Arma grupos de productos curados para destacar en el sitio
- **Organizar Productos**: Define el orden exacto en que aparecerán los productos
- **Probar Versiones**: Crea diferentes versiones (slices) y alterna entre ellas
- **Publicar Rápidamente**: Activa y desactiva colecciones con un solo clic

## Cómo Empezar

Consulta la guía de [Gestión de Colecciones](./collections) para aprender a crear tu primera colección paso a paso.

## Características Principales

### Colecciones Personalizadas
Crea vitrinas con los productos que elijas, organizados como prefieras.

### Slices (Versiones)
Mantén múltiples versiones de la misma colección y alterna entre ellas sin empezar desde cero.

### Activación Instantánea
Los cambios aparecen en el sitio en tiempo real tan pronto como actives un slice.
