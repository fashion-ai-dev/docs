# Importar Productos

Existen dos formas de importar productos a FashionAI.

## Opción 1: importación por CSV

Usa esta opción cuando quieras preparar un archivo manualmente y subir productos en lote.

- [Abrir el área de importaciones CSV](../csv-imports/index.md)

## Opción 2: integración directa

Para algunas tiendas, podemos traer los productos directamente mediante una integración con la plataforma, sin preparar manualmente un archivo CSV.

- [Mira un ejemplo de importación por integración](../integrations/vtex.md)

## ¿Qué opción deberías usar?

- Elige **importación por CSV** si tu equipo quiere control total sobre el archivo antes de subirlo
- Elige **integración directa** si tu catálogo ya se gestiona en tu plataforma y quieres un flujo directo de sincronización
