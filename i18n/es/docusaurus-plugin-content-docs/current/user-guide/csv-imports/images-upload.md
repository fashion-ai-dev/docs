# Carga de Imágenes de Productos

Usa esta opción cuando quieras enviar imágenes de productos por separado de la importación CSV.

## Cuándo es útil

La carga de imágenes es útil cuando:

- tu equipo prefiere gestionar imágenes fuera del archivo CSV
- tu plantilla CSV no exige campos de imagen
- necesitas reemplazar o agregar imágenes de productos en bloque

Puedes cargar imágenes de productos en **Settings** -> **Image Manager**.

## Formatos aceptados

| Formato | Extensión |
|--------|-----------|
| JPEG | .jpg o .jpeg |
| PNG | .png |
| GIF | .gif |
| WebP | .webp |

## Límites

- **Tamaño máximo:** 10 MB por archivo
- **Cantidad:** hasta 500 imágenes por carga

## Regla de nombre del archivo

:::tip Consejo importante
Nombra cada archivo con el **código del producto** seguido de `_01`, `_02` y así sucesivamente. Esto define el orden de las imágenes.

Ejemplo: `SKU12345_01.jpg`, `SKU12345_02.jpg`, `SKU12345_03.jpg`
:::

El nombre del archivo debe terminar con `_01` hasta `_10`.

## Antes de subir

- asegúrate de que el producto ya exista en el catálogo
- confirma que el nombre del archivo usa el código correcto del producto
- revisa formato y tamaño antes de la carga

## ¿Necesitas ayuda?

Si la carga falla, revisa primero los nombres y formatos de los archivos y después contacta a **support@generativecrm.com**.
