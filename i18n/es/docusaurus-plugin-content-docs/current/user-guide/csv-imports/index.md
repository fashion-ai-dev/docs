# Importaciones CSV

Usa importaciones CSV cuando quieras preparar archivos manualmente y subir productos o relaciones en lote.

## Guías disponibles

- [Importación de catálogo de productos](./products.md)
- [Carga de imágenes de productos](./images-upload.md)
- [Importación de Shop The Look](./shop-the-look.md)

## Antes de empezar

- usa siempre la plantilla más reciente descargada del panel
- revisa **Campos de la Hoja de Cálculo** si necesitas ayuda con el formato de los campos
- si los caracteres especiales se ven mal después de exportar, abre el archivo en Google Sheets y vuelve a descargar el CSV antes de subirlo
