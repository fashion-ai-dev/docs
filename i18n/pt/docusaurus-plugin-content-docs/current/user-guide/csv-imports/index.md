# Importações CSV

Use importações CSV quando quiser preparar arquivos manualmente e enviar produtos ou relacionamentos em lote.

## Guias disponíveis

- [Importação de catálogo de produtos](./products.md)
- [Upload de imagens de produtos](./images-upload.md)
- [Importação de Shop The Look](./shop-the-look.md)

## Antes de começar

- sempre use o template mais recente baixado do painel
- consulte **Campos da Planilha** se precisar de ajuda com o formato dos campos
- se caracteres especiais aparecerem errados depois da exportação, abra o arquivo no Google Planilhas e baixe o CSV novamente antes de enviar
