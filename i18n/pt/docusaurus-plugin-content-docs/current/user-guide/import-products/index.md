# Importar Produtos

Existem duas formas de importar produtos para a FashionAI.

## Opção 1: importação por CSV

Use esta opção quando quiser preparar um arquivo manualmente e enviar produtos em lote.

- [Abrir a área de importações CSV](../csv-imports/index.md)

## Opção 2: integração direta

Para algumas lojas, podemos puxar os produtos diretamente por uma integração com a plataforma, sem preparar manualmente um arquivo CSV.

- [Veja um exemplo de importação via integração](../integrations/vtex.md)

## Qual opção usar?

- Escolha **importação por CSV** se seu time quiser controle total sobre o arquivo antes do envio
- Escolha **integração direta** se seu catálogo já for gerenciado na sua plataforma e você quiser um fluxo direto de sincronização
