# Product Image Upload

Use this option when you want to send product images separately from the CSV import.

## When this is useful

Image upload is useful when:

- your team prefers to manage images outside the CSV file
- your CSV template does not require image fields
- you need to replace or add product images in bulk

You can upload product images in **Settings** -> **Image Manager**.

## Accepted formats

| Format | Extension |
|--------|-----------|
| JPEG | .jpg or .jpeg |
| PNG | .png |
| GIF | .gif |
| WebP | .webp |

## Limits

- **Maximum size:** 10 MB per file
- **Quantity:** up to 500 images per upload

## File naming rule

:::tip Important tip
Name each file with the **product code** followed by `_01`, `_02`, and so on. This defines the image order.

Example: `SKU12345_01.jpg`, `SKU12345_02.jpg`, `SKU12345_03.jpg`
:::

The file name should end with `_01` to `_10`.

## Before uploading

- make sure the product already exists in your catalog
- confirm the file name uses the correct product code
- check the file format and size before upload

## Need help?

If the upload fails, review the file names and formats first, then contact **support@generativecrm.com**.
